# NED Chemistry Entry Test Portal

A premium, interactive web application compiled for NED University Engineering Entry Test preparation. This application renders chemical formulas and equations dynamically and quizzes students on a comprehensive shuffled question bank from actual entry-test papers.

## Features

- **Rich Aesthetic User Interface:** Modern, responsive dark mode design with glassmorphism, dynamic gradients, and custom micro-animations.
- **Interactive Practice Modes:** Choose between different quiz modes or target specific sets.
- **Real-Time LaTeX & Chemical Formula Rendering:** Uses MathJax to render complex formulas beautifully (utilizing mhchem).
- **Dynamic Shuffling:** Questions and options are shuffled on every run to ensure effective learning and test simulation.
- **Live Score Tracking:** Real-time feedback on correct/incorrect answers, score percentage, and a comprehensive summary report at the end.
- **Interactive Question Navigator:** Easily navigate through questions, view progress, and jump back to any previous question.
- **Detailed Review Mode:** Allows filtering and reviewing correct, wrong, and skipped questions with highlight notes.

## Getting Started

To run the quiz locally, simply open the `index.html` file in any modern web browser. No installation or build steps are required.
